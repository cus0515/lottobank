# LottoBank 폐쇄 테스트 체크리스트

## 테스터 모집

- [ ] Google 계정 이메일 12개 이상을 확보합니다.
- [ ] 여유 인원까지 포함해 15명 이상을 권장합니다.
- [ ] 테스터에게 14일 동안 참여 상태를 유지해야 한다고 안내합니다.
- [ ] 피드백 수집 채널을 정합니다.

## 테스트 시나리오

- [ ] 회원가입과 로그인.
- [ ] QR 인증 화면 진입.
- [ ] 인터넷 구매 캡처 등록.
- [ ] 회차 조회.
- [ ] 연구소 백테스트.
- [ ] 번호 저장고.
- [ ] 커뮤니티 추천번호 글 작성.
- [ ] 나의 전적 확인.
- [ ] 지도에서 주변 판매점 확인.
- [ ] 다크모드 전환.

## 프로덕션 신청 답변 메모

- 앱의 핵심 가치는 QR 기반 구매 이력과 당첨 전적 관리입니다.
- 테스트 중 확인한 주요 기능은 QR 등록, 전적 계산, 회차 조회, 연구소, 커뮤니티입니다.
- 테스트 피드백은 모바일 UI 정렬, 저장고 버튼, 지도 빈 상태, 스킨 접근성 개선에 반영했습니다.
- 정식 출시 전 개인정보처리방침, 이용약관, 커뮤니티 운영 기준을 확인했습니다.
