• 동행복권 공식 API 데이터를 기반으로 최근 회차 번호 패턴을 분석합니다.

• 인증 데이터 기반: QR 인증 완료 회원의 당첨 이력은 실제 구매 데이터로만 기록됩니다.

• 안내: 번호 분석은 통계적 참고 자료입니다. 당첨을 보장하지 않습니다.
