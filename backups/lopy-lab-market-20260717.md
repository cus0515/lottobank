# LOPY 연구소 기능 백업

이 백업은 연구소 카테고리에 추가했던 LOPY 번호 거래소 기능의 복원 지점을 기록한다.

## Git 복원 지점

- 커밋 해시: `95d8701`
- 커밋 메시지: `feat: add LOPY lab market analysis`

## 포함됐던 기능

- 연구소 상단 `LOPY 차트` 탭.
- LOPY 종합 지수 차트.
- 개별 번호주 미출현 기간 차트.
- 가치 투자, 모멘텀, 볼린저 역추세 추천 조합.
- 무료 기본 설정과 VIP 상세 설정 분리.
- 추천 번호를 연구소 로또 저장고에 저장하는 기능.
